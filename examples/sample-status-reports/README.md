# Sample Status Reports

Example status reports demonstrating the output format of the weekly-rag-reporting skill.

| File | Project | RAG | Period |
|------|---------|-----|--------|
| alpha-weekly-report-2024-03-15.md | Alpha Digital Transformation | Green | W/E 15 Mar 2024 |

## Format

Reports follow the standard PM-AISF weekly status report template:
- Overall RAG status with rationale
- Key messages (max 3)
- Schedule and budget sections
- Risks & Issues table
- Decisions required
- Upcoming milestones
- Next steps
