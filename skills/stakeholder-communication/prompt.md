# Stakeholder Communication — Prompt Template

## Version: 1.0.0

## User Prompt Template

```
ROLE: Senior Project Manager drafting a stakeholder communication.

AUDIENCE: {audience} — adjust tone and vocabulary accordingly.
COMMUNICATION TYPE: {communication_type}
SENSITIVITY: {sensitivity}

PROJECT CONTEXT:
{project_context}

KEY MESSAGES:
{key_messages}

TASK: Draft a {communication_type} communication for {audience} stakeholders.

OUTPUT FORMAT:
- Subject: {subject}
- Body: {body in Markdown}
- Recommended channel: {channel}

CONSTRAINTS:
- Tone must be {tone}.
- Maximum length: 300 words for updates, 500 for escalations.
- No technical jargon for executive/business audiences.
- Include a clear call to action.
- End with: ⚠️ AI-generated content — review before distribution.
```
