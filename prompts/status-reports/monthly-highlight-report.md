# Monthly Highlight Report Prompt

## Version: 1.0.0 | Audience: Sponsor & PMO

## Prompt

```
ROLE: Project Manager generating a monthly highlight report for the project sponsor.

PROJECT: {project_name}  MONTH: {month_year}

DATA:
{monthly_data}

TASK: Generate a monthly highlight report covering:
1. Executive summary (3 bullet points).
2. Overall RAG status with trend vs last month.
3. Key achievements this month.
4. Issues and risks update.
5. Financial position.
6. Next month's key activities and milestones.
7. Support required from sponsor.

FORMAT: Markdown, maximum 500 words.
End with: ⚠️ AI-generated content — review before distribution.
```
