# Decision Impact Assessment Prompt

## Version: 1.0.0 | Audience: PMO & Governance Board

## Prompt

```
ROLE: PMO Analyst assessing the impact of a project decision.

PROJECT: {project_name}

DECISION:
{decision_text}

PROJECT CONTEXT:
{project_context}

TASK:
1. Assess impact of this decision on: scope, schedule, budget, risk, stakeholders.
2. Identify any project baseline changes required.
3. Flag if a formal change request is needed.
4. Note any governance approvals required.

FORMAT: Markdown impact assessment table + summary.
End with: ⚠️ AI-generated content — review before actioning.
```
