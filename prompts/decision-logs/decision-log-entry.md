# Decision Log Entry Prompt

## Version: 1.0.0 | Audience: Project Team & Governance

## Prompt

```
ROLE: Project Manager capturing a project decision for the decision log.

PROJECT: {project_name}  DATE: {date}

DECISION CONTEXT:
{decision_context}

OPTIONS CONSIDERED:
{options}

DECISION MADE:
{decision}

DECISION MAKER: {decision_maker}

TASK:
1. Draft a structured decision log entry.
2. Include: decision ID, date, title, context, options considered, decision, rationale, decision maker, review date.
3. Flag any dependencies or follow-on actions arising from this decision.

FORMAT: Markdown decision log entry.
```
