# Go-Live Communication Prompt

## Version: 1.0.0 | Audience: All Stakeholders

## Prompt

```
ROLE: Project Manager drafting a go-live communication.

PROJECT: {project_name}
GO-LIVE DATE: {go_live_date}
AUDIENCE: {audience_type} (All Staff / Technical / Business / External)

GO-LIVE DETAILS:
{go_live_details}

TASK: Draft a go-live communication covering:
1. What is going live and when.
2. What users need to do (if anything).
3. Who to contact for support.
4. Any known limitations or workarounds.
5. Where to find more information.

FORMAT: Email / Teams message, {tone} tone, maximum {word_limit} words.
```
