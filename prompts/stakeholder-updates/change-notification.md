# Change Notification Prompt

## Version: 1.0.0 | Audience: Affected Stakeholders

## Prompt

```
ROLE: Change Manager drafting a change notification.

CHANGE: {change_title}
EFFECTIVE DATE: {effective_date}
AUDIENCE: {affected_groups}

CHANGE DETAILS:
{change_description}

IMPACT:
{impact_summary}

TASK: Draft a change notification covering:
1. What is changing and why.
2. Who is affected and how.
3. What action is required from the audience.
4. Timeline and key dates.
5. Where to get help or raise concerns.

FORMAT: Email, formal tone, maximum 400 words.
```
