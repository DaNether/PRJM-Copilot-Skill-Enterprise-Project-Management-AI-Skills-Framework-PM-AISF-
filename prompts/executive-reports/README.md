# Executive Reports — Prompt Library

Prompts for C-suite, board, and programme governance reporting.

| File | Audience | Description |
|------|----------|-------------|
| portfolio-dashboard.md | CEO / Board | Portfolio-level health and investment summary |
| programme-board-pack.md | Programme Board | Structured programme board meeting pack |

## Usage

1. Copy the prompt template from the relevant file.
2. Replace `{variables}` with your project data.
3. Submit to your AI model (Copilot Studio, Azure OpenAI, etc.).
4. Review the output before distribution.
