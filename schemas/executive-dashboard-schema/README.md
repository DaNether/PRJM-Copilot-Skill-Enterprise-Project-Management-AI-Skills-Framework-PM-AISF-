# Executive Dashboard Schema

**Version:** 1.0.0 | **Standard:** JSON Schema Draft 2020-12

Defines the portfolio-level executive view structure used by the Executive Reporting Agent and Steering Committee Agent.

## Usage

This schema is used as:
1. The output contract for the `executive-reporting-agent`.
2. The input/output contract for the `steering-committee-agent`.
3. An Azure OpenAI function definition for portfolio dashboard tools.
